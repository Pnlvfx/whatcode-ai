# About

My name is Simone Gauli. I'm an Italian developer with 6+ years of experience, mostly working as a freelancer.

I built WhatCode because I kept running into the same frustration: I'd kick off an OpenCode session and walk away from my machine, with no good way to know when the agent finished or what it was doing. I tried SSH, Telegram bots, and a few other "solutions" all of them were painful to set up, awkward to use, and felt nothing like a real app.

So I built one.

WhatCode is a solo side project I started in my free time and decided to publish. The source code is [open on GitHub](https://github.com/Pnlvfx/whatcode-ai) — you can see exactly what the daemon does and how the app communicates with your machine.

If you have questions, feedback, or just want to say hi, find me on X: [@simone_gauli](https://x.com/simone_gauli).
